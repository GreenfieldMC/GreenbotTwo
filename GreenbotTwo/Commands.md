## Install Command

How to install the Greenfield project. Two links are configured in the appsettings.json file to specify a Windows download and a Mac download.

#### Usage
`/install operating_system [for_user]`

#### Permissions
`GreenbotTwo.Install` - Allows a user to run the command for themselves.
`GreenbotTwo.Install.Others` - Allows a user to run the command for other users.
